# Contributing to Arabic RTL Fixer 🤝

شكراً إنك عايز تساهم! ده بيساعد كل الناس اللي بتشتغل بالعربي على الإنترنت.

---

## 🐛 الإبلاغ عن مشكلة / Reporting a Bug

1. افتح [Issue جديد](../../issues/new)
2. اكتب:
   - اسم الموقع اللي فيه المشكلة
   - Screenshot لو ممكن
   - Chrome version بتاعك

---

## 💡 اقتراح ميزة / Suggesting a Feature

افتح Issue وابدأ العنوان بـ `[Feature]`، واشرح الفكرة.

---

## 🔧 تقديم كود / Submitting Code

```bash
# 1. Fork الـ repo
# 2. Clone
git clone https://github.com/[your-username]/arabic-rtl-fixer

# 3. اعمل branch جديد
git checkout -b feature/my-feature

# 4. اعمل تغييراتك
# 5. Test في Chrome (Load unpacked)

# 6. Commit
git commit -m "feat: add Hebrew support"

# 7. Push وافتح Pull Request
git push origin feature/my-feature
```

---

## 📐 معايير الكود / Code Standards

- كومنتات عربي أو إنجليزي، كلاهما مقبول
- لا تضيف external dependencies
- كل حاجة لازم تشتغل locally بدون سيرفر

---

## 🧪 كيفية الاختبار / Testing

1. روح `chrome://extensions`
2. فعّل Developer mode
3. Load unpacked واختار الفولدر
4. جرّب على مواقع زي: Wikipedia عربي، Twitter، أي مدونة عربية

---

شكراً! 🙏
